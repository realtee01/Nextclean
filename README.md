# Nextclean. — Redefining Domestic Excellence 🧹✨

Nextclean is a premium, high-converting landing page built for a modern cleaning agency based in Lagos, Nigeria. The project focuses on a "Luxury-Minimalist" aesthetic, utilizing smooth micro-interactions and a mobile-first responsive layout to provide a seamless user experience for both local and diaspora clients.

**Live Demo:** [nextclean.vercel.app](https://nextclean.vercel.app)

---

## 🚀 Key Features

* **Dynamic Service Filtering:** Smooth category switching (Residential, Commercial, Deep Clean) using Framer Motion's `layout` and `AnimatePresence`.
* **Diaspora-Centric Testimonials:** Curated social proof targeting Nigerians in the UK and US who manage properties back home.
* **Fully Responsive Design:** Custom-tailored layouts for mobile, tablet, and desktop viewports to ensure zero "clashing" of elements.
* **Interactive Booking UI:** A bouncy, spring-physics-based CTA system that triggers a clean, focused Contact Modal.
* **Modern Tech Stack:** Built with performance and scalability in mind using Vite and React.
